Join:

You browsed through the events clicked a few to see their content and are now wondering how to join them as the join button is deactivated.

To join an event the following points have to be considered:

  • You need to be logged in to join an event.
  • You can only join an event with one character.
  • Are you the game master of this event? Because you can either be a player or the game master.
  • The current amount of players does not exceed the maximum amount of player slots for the event. Look at "Max. Player" on the event to see how many players are allowed to play the game.
  • Is the event expired? If it is then that's why you can't join.

If all of that is fine the join button will be active. :-)

Pressing the button will show you a list of your character, from which you can pick one to join the game.

Don't worry if you made a mistake, you can remove your character from the event in the section below and add the correct character.


Like:

If you think this event deserves positive feedback, consider liking the event. For that, you will find in the event details the option to "Like" right above the character section.
Click on the heart symbol to give the event a like or to remove your previously given like.

All events can be "liked" but an event can only be liked once unless you "unliked" it(then you can "relike").


Comment:

If you are logged in, you can leave comments on every event you like.
The general rules here are to refrain from spamming, be respectful, and keep on topic.
Please always make sure that your comment is related to the event and TTRPGs, D&D, etc. in general.



Create An Event
Click to see more
Minimal Requirements to create an Event:

You would like to have your event displayed on our blog? Well, Thank you that is awesome! :-)

Once you created an account and logged in the option to create an event will be on the navigation bar.
To make it easy, most fields in the event creation form are optional and can be updated later.

You only need to fill in:

Title
Start Date

All other fields are optional or have a default value.

However the default value of "Status" is "Private"
That means you would create a privat event

Status:
"Private"

Private events are not shown on the main page nor displayed in the search results.
You can only find them in your profile under "Owned Events".

"Public"

Public events are on the main page and displayed in the search results.
You can also find them in your profile.

"Expired"

Expired events are not shown on the main page but are displayed in the search results.
You can also find them in your profile.

Exceptions

There might be some expired events on the main page. These are events where the start date is smaller than the current date.
That caused them to get displayed as "expired", internally their "Status" is still "Public".
This is a bit of a grace period for events. Once the status has been updated, the event will no longer be on the main page.

Background Snippet:

The background snippet is what you see on the main page by the event.
A small little excerpt or summary of the event, to give people some idea, of what to expect.

The standard value is: "Chick the event title above for more..."
So you might want to replace that with your event background snippet.

Max. Player:

Max. Player id is the maximum number of players that can join the event.
The standard value is "4".
The minimal value is "2".
The maximal value is "6".

A Game Master could add more than 6 to an event, by updating the event and adding the character there.
However, this is usually ill-advised.

Optional fields:
"Background"

While optional, it is advised to write a short or long summary of your event or episode of your event.
You can upload pictures and videos as well in this field.

"Image"

Your event should have an image that you selected, as it will otherwise just get a boring default image until you update it.
Please only use images that are appropriate and related to the event and TTRPGs, D&D, etc. in general.
Preferably use pictures with 450 (width) x 250 (height) pixels.

"Tags"

Tags will help people find your event by categorizing it with appropriate tags.
It also gives the visitor an idea of what kind of event to expect.

"Main Link"

This will later be a big button on the event page that directs visitors to your webpage/channel to what the event is.
So if you already have a link for that, you should put it in this field.
It is an URL-Form-Field, so you need to use something like "https://www.twitch.tv/". Starting just with "www" will throw an error.

"Shoutouts"

If you want to make special announcements, shoutouts to artists, music composers, and so on, you can do that there.

"Other Links"

If your players have their webpages, etc. you can show the links here.

"Game Master"

If you already have a game master for your event, you can add the username here.
Please note that being a game master for your event does not require you to be a game master on RND|Roll.

"Ruleset"

Add the rule set for your event here.
D&D 5e, Pathfinder 2e, Homebrew, How to be a Hero, and so on.

Game Master and Administrators

Click to see more
The Administrators Role

If major problems arise the Site Administrators can help out by editing or removing nearly any content on RND|Roll.
However, this should only be done with exceptions.
To contact an Administrator please use the "Message"-Function next to "Logout" on the navigation bar.

What Is A Game Master

Generally speaking, the RND|Roll Staff likes to be more hands-off and let most stuff be sorted out by the user.
Therefore a user can be promoted to "Game Master".
Game Masters can update any event and remove players from it.
If the event owner is currently unavailable and updates to an event have to be done, Game Master can therefore be contacted to take care of this.
To contact a Game Master please use the "Message"-Function next to "Logout" on the navigation bar.

Can I Be Promoted To Game Master

Every player can request the promotion to Game Master in their profile.
Now just because it was requested, does not mean that promotion will happen, because there is a bit of reviewing going on.
If there are already enough game masters around, any future promotion may be set on hold.
Please keep in mind that being a game master is a great responsibility and appropriate conduct is expected.

How To Create A Profile
Minimal Requirements to create a Profile:

If you created your account and logged in, you will see the option to create a Profile on the navigation bar.
The minimal requirements to create a profile are:
1. Login in with your account
2. Clicking on create a profile
3. Click on "Create Profile"

Done! You have a Profile.

If you want to update your Profile, you will find the option for that in your Profile.

Optional Fields for the Profile:
"Image"

Since some users may visit your profile to see what events or characters your made and participated in, you might want to give a good or at least memorable impression. :-)
Therefore it is a good idea to add an image to your profile, something that goes well along with your username is often a good idea.

"Bio"

Tell people a bit about yourself or your channel/ website.

"Links"

If you have Websites, Facebook, Twitter, discord, and so on, put the links there and they will be on your profile.

What Are They Good For?

Well, beside being the only place to find "Private Events" and "Create Characters"
Profiles are a good place to have an overview of your created events, participated events, and characters. Any authenticated user can visit your profile and view your events and characters.

You can edit your profile, and your user settings and change your password on the profile page. You can also request a promotion to Game Master here.

If you are a Game Master or Administrator, you can read the messages from the users here.

Minimal Requirements to create a Character:

The minimal requirements to create a character are:

Name
Class

That is all you need to create a character.

Optional Fields for the Character:
"Image"

Here you can upload a nice picture for your character.
Preferably use pictures with 150 (width) x 200 (height) pixels.

"Excerpt"

Put a nice quick summary or an excerpt of the character backstory, to tell everyone what your character is about.

"Background"

Feel free to put in a whole lot of background if you want to.

"Items"

If you already know what he is going to have in his bag, then you can add some items here.

However, anything that he has already equipt, has to be put in the equipment.
This can be edited on the character detail page later.

You can update your character on the character detail site.
You can access this site by clicking on the character picture of the character in your profile.


Character Page

On the top of the character page, you have the option to edit your character or delete it.

"Equipment"

A bit below you have a dropdown for the equipment.
There is also an option to edit your equipment.

"Notes"

"Joined Events"

a list of all events this character participated in can be found as well.
This way interested visitors can see which events they want to check out as well as if they like your character.
